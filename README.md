### Juan Pablo Medina | Universidad del Cauca — 2026

---

## Descripción

Sitio web personal desarrollado como portafolio y currículum vitae en línea. El proyecto fue implementado en *dos versiones* con el mismo contenido, para comparar los enfoques de desarrollo web *MPA* (Multi-Page Application) y *SPA* (Single-Page Application).

---

## Estructura del proyecto

```
/
├── MPA/                       
│   ├── index.html
│   ├── css/
│   │   └── styles.css
│   ├── js/
│   │   └── scripts.js
│   └── pages/
│       ├── about.html
│       ├── estudios.html
│       ├── pasatiempos.html
│       ├── proyectos.html
│       └── contacto.html
│
└── SPA/                      
    ├── index.html
    ├── css/
    │   └── styles.css
    └── js/
        └── scripts.js
```

## Instrucciones de uso

*Versión MPA:*
1. Abre `MPA/index.html` en tu navegador.
2. Navega entre secciones usando la barra de navegación superior, cada clic carga una página nueva.

*Versión SPA:*
1. Abre `SPA/index.html` en tu navegador.
2. Usa la barra de navegación o simplemente haz scroll para recorrer las secciones.
